WWW as a Rube Goldberg machine

Proposal

This project aims to provide users with the ability to explore and discover random meals, search for specific recipes. When the user clicks the food image, random cocktail will be recommended.

Features

Search Functionality: Users can input a meal name into the search bar. And a search button triggers the search functionality. Also, the search results include the meal's name, image, and ingredients.

Random Meal Generator: A "Eat What Today?" button fetches a random meal from the MealDB API. It displays information of meal's name, image, and ingredients.

Visual Appeal: Headers and list items have a clean and easy-to-read style.


The web page is designed to be responsive, ensuring a seamless experience across various devices.
